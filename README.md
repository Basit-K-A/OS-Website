# portfolioOS

A single-page portfolio that mimics a minimalist Arch Linux / i3 desktop. Navigate via the terminal, menus or icons. Resize, move and fullscreen windows.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

Open https://minimialist-website.vercel.app/

## Terminal commands

- `1`–`5` or `about`, `projects`, `blog`, `experience`, `contact`
- `help` — list commands
- `clear` — reset terminal output
- Press `/` to focus the input
